# ``AnvyxCipherBiometric``

Biometric-gated keys and secrets — Face ID / Touch ID protection via the Secure
Enclave and Keychain access control.

## Overview

Store keys and secrets that can only be used after a successful biometric
authentication, backed by `kSecAccessControl` / the Secure Enclave.
